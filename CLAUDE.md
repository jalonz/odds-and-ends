# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Overview

A public collection of personal projects. Each project lives in its own top-level directory and is self-contained — no shared build system, no cross-project dependencies. When adding a new project, update the project index below and add a README to the project directory.

## Conventions

- Each project directory is standalone: its own README, assets, and any tooling it needs
- No secrets, credentials, or personal data — this repo is public
- Keep the root README updated with a link to each project

## Project Index

- `japanese_villages/` — Interactive era-by-era index of Japanese village economy flourishing (~14,000 BCE to present), built as reference material for a game design project
- `fidelity-to-markdown/` — CLI ETL pipeline that converts Fidelity "Positions" CSV exports into LLM-ready markdown tables for portfolio analysis
